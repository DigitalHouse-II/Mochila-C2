A. ¿Quién inventó el sistema de control de versión Git y por qué? R// Git fue diseñado por Linus Torvalds con el proyecto de Linux y el workflow  de sus desarrolladores en mente y, siendo estos los primeros en adoptarlo como sistema de control de versiones. El motivo por el cual fue creado fue para dar seguimiento de todos los cambios individuales realizados por cada colaborador que al contribuir evitar que el trabajo concurrente entre en conflicto.
