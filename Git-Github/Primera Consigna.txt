•¿Quién inventó el sistema de control de versión Git y por qué?
    El sistema de control de versiones git fue creado por Linus Torvalds en 2005 para ser utilizado en el desarrollo del kernel de Linux. Git es un software escrito en C, con interfaz por lí­nea de comandos, disponible para Windows, MacOS X y GNU/Linux.

•¿A quién pertenece actualmente Github y por qué?
    Desde enero de 2010, GitHub opera bajo el nombre de GitHub, Inc. El 4 de junio de 2018 Microsoft compró GitHub por la cantidad de 7500 millones de dólares.
    Desde el nombramiento de Satya Nadella como CEO de Microsoft, la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios.

•¿Hay otra forma que no sea la terminal para trabajar con Github?
    Si, se puede trabajr con la interfaz grafica de la página web de modo que se pueden crear repositorios, manejar y crear ramas, realizar commits y cambios y finalmente, realizar pull requests y merging.
