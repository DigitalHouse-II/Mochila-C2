¿Quién inventó el sistema de control de versión Git y por qué?
Git fue diseñado por Torvalds con el proyecto del kernel de Linux y el workflow de sus desarrolladores en mente y, obviamente, estos fueron los primeros en adoptarlo como sistema de control de versiones

¿A quién pertenece actualmente Github y por qué?
El 4 de junio de 2018 Microsoft compró GitHub por la cantidad de 7500 millones de dólares y actualmente ellos son los dueños

¿Hay otra forma que no sea la terminal para trabajar con Github?
Se puede trabajar con GitHub desde la nube para crear archivos sin embargo no se pueden crear folders y no quedaría guardado el repositorio localmente.
