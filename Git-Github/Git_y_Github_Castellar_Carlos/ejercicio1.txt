a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git fue diseñado por Linus Torvalds Git fue creado en el 2005, porque la relación entre la comunidad que desarrollaba el kernel de Linux y la compañía que desarrollaba BitKeeper se vino abajo y la herramienta dejó de ser ofrecida de manera gratuita. Esto impulsó a la comunidad de desarrollo de Linux (y en particular a Linus Torvalds, el creador de Linux) a desarrollar su propia herramienta basada en algunas de las lecciones que aprendieron mientras usaban BitKeeper. Así nace git como software de control de versiones, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. By: Germán Esobar.

b. ¿A quién pertenece actualmente Github y por qué?
Github es actualmente propiedad de Microsoft, esto es el resultado del acercamiento cada vez mayor de Microsoft hacia el desarrollo open source y la comunidad de desarrollo libre, es una estrategia que busca acercarse a la competencia para ver sus necesidades y ofrecer herramientas.
